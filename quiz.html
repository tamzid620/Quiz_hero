2.what is the difference between localStorage and sessionStorage?

A: LocalStorage-localStorage is a persistent storage that allows web applications to store data with no expiration date. sessionStorage-sessionStorage is a temporary storage that allows web applications to store data for a single session or browsing session.

3.what is event loop? how does event loop works?

A: Event Loop-An event loop is a programming construct that enables the execution of non-blocking I/O and asynchronous tasks in a program.
The event loop works by continuously monitoring a queue of tasks or events, and executing them in the order they are received. When a task is received, the event loop adds it to the queue and moves on to the next task.
